# useStorageAt

Hook for returning the value from a storage slot at a given address.

## Import

```ts
import { useStorageAt } from 'wagmi'
```

## Usage

::: code-group
```tsx [index.tsx]
import { useStorageAt } from 'wagmi'

function App() {
  const result = useStorageAt({
    address: '0xFBA3912Ca04dd458c843e2EE08967fC04f3579c2',
    slot: '0x0',
  })
}
```
