# useCallsStatus

Hook to fetch the status and receipts of a call batch that was sent via [`useSendCalls`](/react/api/hooks/useSendCalls).

 

## Import

```ts
import { useCallsStatus } from 'wagmi'
```

## Usage

::: code-group
```tsx [index.tsx]
import { useCallsStatus } from 'wagmi'

function App() {
  const result = useCallsStatus({
    id: '0x...', // [!code focus]
  })
}
```
