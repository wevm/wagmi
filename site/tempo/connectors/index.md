# Connectors

Wagmi-specific connectors for Tempo.

## Import

::: code-group
```ts [React]
import { tempoWallet } from 'wagmi/tempo'
```

```ts [Core]
import { tempoWallet } from '@wagmi/core/tempo'
```
:::

## Available Connectors

- [**`tempoWallet`**](/tempo/connectors/tempoWallet) Connector for the Tempo Wallet
- [**`webAuthn`**](/tempo/connectors/webAuthn) Connector for a WebAuthn EOA.

:::tip
See the [Tempo Accounts SDK docs](https://docs.tempo.xyz/accounts) for more info on how `tempoWallet` and other connectors work.
:::
