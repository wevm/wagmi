# Connectors

Wagmi-specific connectors for Tempo.

## Import

```ts
import { tempoWallet } from 'wagmi/tempo'
```

## Available Connectors

- [**`tempoWallet`**](/tempo/connectors/tempoWallet) Connector for the Tempo Wallet
- [**`webAuthn`**](/tempo/connectors/webAuthn) Deprecated. Use `webAuthn` from `accounts/wagmi` instead.
- [**`dangerous_secp256k1`**](/tempo/connectors/dangerous_secp256k1) Deprecated. Use `dangerous_secp256k1` from `accounts/wagmi` instead.
